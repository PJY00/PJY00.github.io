100번하고 1001번은 같은 문제라고 봐도 무관하니까 같이 올립니다.

1000번 A+B 문제

```c
#include <stdio.h>

int main(){
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a+b);
    return 0;
}
```

1001번 A-B문제

```c
#include <stdio.h>
int main(){
    int a, b;
    scanf("%d %d", &a, &b);
    printf("%d", a-b);
    return 0;
}
```
